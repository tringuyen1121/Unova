# Unova: Auto attendance check for students
Unova is the application used by students to check their attendance at universities or schools. The app also provides many other
utilities for students such as showing the progress of taken courses, schedule, finding classmates, ...

## Requirement and Dependencies
- Minimum Version: iOS 3.0 
- iBeacons
- Firebase

## Installation
### 1. Clone or download this repo.
`git clone https://github.com/tringuyen1121/Unova.git`
### 2. Import project into XCode
Import project into XCode and install all requirements
### 3. Deploy the application
Connect an iPhone that meet the requirements. Run app from XCode and choose your phone. It is recommended to test the app on a real phone for fully function LE Bluetooth.
### 4. Further development
To modify the app, you need to change the uuid of beacons used in the app to yours, and also change the configuration to use your Firebase.
